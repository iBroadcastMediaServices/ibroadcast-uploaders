# ibroadcast-uploader.py
Move this script to the root directory of your music files and then run the following commands:
```
python3 -m venv ibroadcast
$ source ./ibroadcast/bin/activate
$ pip install -r requirements.txt
$ python3 /ibroadcast-uploader.py
```

When your uploads are complete, you can deactivate the venv:
```
$ deactivate
```
